    char 변수명 = 'a';
    - 문자 배열을 이용한 문자열 만들기
        char[] 배열변수 = new char[3];
        char[0] = '유';  char[1] = '재';  char[2] = '석';

[2] 자바가 문자열을 다루는 방법 = String 클래스 지원
     1. String 변수명 = new String("문자열");
     2. String 변수명 = "문자열";

[3] 문자열 연결하는 방법
    1. 문자열1.concat( 문자열2 )
    2. 문자열1 += 문자열2
    3. StringBuilder 변수명 = new StringBuilder();
       변수명.append( 문자열 );
    차이점 : 새로운 문자열 반환 차이.
        - StringBuilder 클래스는 기존 메모리를 그대로 연결해서 사용한다.

[4] String 문자열에서 자주 사용하는 함수들
    1. 문자열.charAt(인덱스)                           : 해당 인덱스번호의 문자 1개 반환 함수
    2. 문자열.length()                                : 문자열의 길이 반환 함수
    3. 문자열.replace( 기존문자열 , 변경할문자열 )        : 문자열내 기존문자열이 존재하면 변경할문자열로 치환/교체 된 새로운문자열 반환 함수
    4. 문자열.subString( 시작인덱스 , [끝인덱스] )       : 문자열을 인덱스 기준으로 잘라낸 새로운 문자열 반환
    5. 문자열.split("구분 문자")                       : 문자열내 구분문자 기준으로 분해 해서 새로운 문자열 배열 반환
    6. 문자열.indexOf("찾을문자열")                     : 문자열내 찾을문자가 존재하면 찾은문자열의 인덱스 반환 , 없으면 -1 반환
    7. 문자열.contains("찾을문자열")                    : 문자열내 찾을문자가 존재하면 true 반환 , 없으면 false 반환
    8. 문자열.getBytes()                              : 문자열내 문자 하나씩 바이트로 변환된 바이트 배열로 반환

[5] 문자 와 바이트 관계 , 자바는 한글/전세계공용어 도 표현가능한 유니코드, UTF-8 사용한다.
    - 모든 문자는 아스키코드 규약/규칙 기준으로 바이트를 문자로 표현
    - 97(10) --> a   , 98(10) --> b , 44032 --> 가
    - char타입 : ''작은따옴표감싼타입 , 문자1개 , 65536글자 표현 , 2바이트 , ( unsigned )

- 함(상자) 수(숫자) : 숫자/코드 가 들어있는 상자
    - 미리 정의한 코드가 있는 구역
    - 매개변수 : 미리 정의한 구역에 들어갈 변수들
    - 리턴 : 미리 정의한 코드의 결과 값을 반환
